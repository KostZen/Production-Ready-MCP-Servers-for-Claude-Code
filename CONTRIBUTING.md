# Contributing

## How to suggest an MCP server

Contributors can suggest a new MCP server by opening an issue or pull request.

Please keep suggestions practical. This catalog is focused on MCP servers that are useful for Claude Code, Cursor, and AI coding agents in real workflows.

## Required information

- Name
- GitHub or docs link
- Category
- Short description
- Install command or config example
- Why it is useful
- Current status: Verified / Popular / Experimental / Needs review

## Quality rules

- Do not add dead or archived projects without marking them
- Do not mark a server as Verified without checking docs and installation
- Prefer practical, production-ready tools
- Add TODO when information is missing
- Keep descriptions short and useful

## Status meaning

- Verified: The server has an active repository, clear docs, and installation instructions that were checked.
- Popular: The server is widely used or officially maintained, but still needs a fresh install check before being marked Verified.
- Experimental: The server looks useful but is early, unstable, or intended for testing.
- Needs review: The server has not been checked enough yet, or the exact repository/docs/install link is missing.
